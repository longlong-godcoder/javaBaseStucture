==================================================================
                            动态规划
==================================================================
1.主要解决最优化问题
2.如果比较初级，思路是这样的：
a) 寻找暴力递归解决方案（自顶向下，这里会有很多重叠解决的问题）
b) 记忆化搜索，记录已经计算过的结果，做到优化
c) 递推，自底向上

ps：所以完整的动态规划，就是将低效率的递归问题转化为优化过的递推问题。

3.相对高级的，思路是这样：
a) 动态的本质，是会变化的状态，所以首先要定义状态（原问题与子问题共同的解）
比如自CoinChange中的dp[i]
b) 定义初始状态（边界）
比如dp[1]
c) 定义状态转移方程
比如dp(i) 和 dp(i - 1)的关系

4.总结而言，简单的说，
动态规划就是把愿问题分解成若干个步骤的子问题，每个子问题的仅仅解决一次并，并保存下来给下一个子问题

5.什么问题显而易见的可以用动态规划来解决呢？
a）子问题存在最优解
b) 无后效性，简单的说，就是前面状确定后不变，不会随着后面状态的变化而变化
